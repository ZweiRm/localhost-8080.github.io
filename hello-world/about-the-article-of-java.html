# 一点点想法

整理笔记的时候，顺便在构想要写多详细，最后得到了这样的结论😗：

毕竟是笔记嘛，那就要全面，但不是写成 Wiki，一是水平不够，二是大把资源已经是现成的了。

然后呢就不能太精简，写的内容全都是特别容易忽略的那种针对性很强的 tip，就有种只是面向我自己用，意义不太大。

但是整理着发现，内容十分杂乱，可能后序还需要慢慢调整一下结构顺序。
