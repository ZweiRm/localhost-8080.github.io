# Kotlin 概述

# 关于 Kotlin

  • 2011 年，JetBrains 公司在 JVM 语言峰会 (JVM Language Summit) 上公布了他们基于 JVM 的静态语言 Kotlin。

  • Kotlin 与 Java 100% 兼容，它可以运行在 Java 虚拟机、Android和浏览器上。

  • 版本变化

    • 1.0 2016 年 2 月
    • 1.1 2017 年 3 月
    • 1.2 2017 年 11 月
    • 1.3 2018 年 10 月
