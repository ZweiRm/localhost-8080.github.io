## 内容

### 编程语言
- **Java** — 语法、面向对象、API 类库、JVM
- **Kotlin**
- **Python** — 语法基础
- **C** — 常量与数据类型

### 后端开发
- **Web** — HTTP 协议
- **Spring** — IoC 等核心概念
- **MySQL** — 事务、索引、锁、分库分表

### 学科基础
- **算法** — 基础算法、动态规划
- **软件工程** — 软件过程

### 机器学习
- **人工智能** — 搜索策略

- **推荐系统**
- **数据分析与人工智能定量分析**

### 大数据

### 杂谈
- Computing Machinery and Intelligence、学术研究方法、博客搭建、服务器配置等
